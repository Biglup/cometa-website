Why Choose Cometa?

Engineered for Excellence

Built upon the robust, MISRA C compliant, and rigorously tested libcardano-c. Inherit commercial-grade quality, security, and reliability for your Cardano applications.

Performance & Efficiency

Leverages the speed of C99 and lightweight design principles. Bindings like Cometa.js use embedded WASM for minimal footprint and fast execution, without special server setups.

Idiomatic & Easy Integration

Cometa aims for natural, idiomatic bindings in each language, making Cardano development feel familiar and accelerating your workflow.

Cross-Platform & Thread-Safe

Based on `libcardano-c`'s design, Cometa bindings are built to be cross-platform and friendly to multi-threaded applications thanks to no shared global state.

Comprehensive Features

Access core Cardano functionalities like address generation/parsing, Ed25519 cryptography, transaction building, serialization/deserialization, and more.

Permissively Licensed

Uses the Apache 2.0 License, allowing broad use including commercial applications, modification, and distribution.

Getting Started

Ready to build? Choose your preferred language binding from the list above, follow its specific installation guide (usually just a package manager command!), and start coding.

Check the README in each binding's repository for minimal examples, like generating an address, to get you started quickly.

Under the Hood: Quality & Design

Layered Architecture

libcardano-c provides distinct layers, offering both fine-grained control for power users and convenient abstractions for common tasks.

Flexible Memory Model

Designed explicitly to facilitate seamless bindings with diverse programming languages and their memory management paradigms.

No Runtime Dependencies*

Core dependencies like libsodium and libgmp are statically linked, simplifying deployment for users of Cometa bindings. (*Bindings themselves may have language-specific deps).

WASM for the Web

Bindings like Cometa.js embed the core logic into a WASM module, enabling high performance and easy distribution in JavaScript environments.

Rigorous Standards

Adherence to MISRA C guidelines and comprehensive unit testing ensure the reliability and security demanded by mission-critical applications.

UTF-8 Handling

Properly handles UTF-8 encoding throughout the library, crucial for metadata and identifiers.
